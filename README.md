# AdventOfCode
All of my Advent of Code puzzle solutions from over the years, consolidated into a single .NET 6 project.  

Project overview
------
Puzzle solutions 

Command line usage
-------
```
advent <year> [<day> [<part> ["<input text>"]]]
```

Example usage
-------
```
advent 2016
advent 2016 1
advent 2016 1 1
advent 2016 2 1 "LLLDDRRR"
```
